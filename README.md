# HorizonIoT-RL
A reinforcement learning framework for long-horizon management of resource-constrained IoT sensing networks in the context of structural health monitoring

The framework supports energy-aware sensor scheduling, battery sustainability analysis, service-quality preservation, feasibility-aware control, and recurrent decision-making under partial observability.

## Key Features

- Long-horizon IoT sensor scheduling environment
- Battery-aware and service-aware reward design
- Support for PPO, DQN, Recurrent PPO, and Maskable PPO
- Feasibility-aware action masking for resource-constrained operation
- Evaluation metrics for service quality, battery sustainability, and constraint satisfaction
- Reproducible configuration files for experiments in varied application domain
